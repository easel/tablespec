## Current-State Summary

### Observed Product Behavior

tablespec is a Python library providing type-safe Pydantic models for the Universal Metadata Format (UMF), a YAML-based schema format for healthcare data tables. It generates SQL DDL, PySpark schemas, and JSON Schemas from UMF; integrates bidirectionally with Great Expectations for validation; converts profiling results from Spark and Deequ into UMF; and generates structured LLM prompts for AI-assisted schema enrichment.

### Observed Architecture and Runtime Shape

Layered architecture with UMF Pydantic models at the core. Type mappings serve as a central hub. Schema generators, GX integration, profiling mappers, prompt generators, and validators are independent modules. PySpark is an optional dependency isolated to two modules with conditional imports.

### Operational and Delivery Context

- Distributed via GitHub Pages PyPI index (not public PyPI)
- Three CI pipelines: test coverage, pre-commit quality, release/deploy
- Release triggered by git version tags
- Development uses uv, ruff, pyright, pytest, pre-commit
